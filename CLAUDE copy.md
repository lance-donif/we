## 公众号文章写作指南

本文档定义以**身体感受与自我照料**为核心主题的公众号文章写作规范。内容覆盖睡眠、身体信号、情绪感受、日常仪式、关系空间等多元维度，写法混合观察诗意、冷知识、体验共鸣、观念重塑、读者故事、数据洞察等。

**同时满足**：

1. **搜索友好**：标题清晰直接，正文有科学依据或心理学视角，排版层次分明。
2. **情感信任**：不制造焦虑，温柔接纳已发生的事，陪读者看见并理解身体与情绪发出的信号。

**账号定位**：新手期个人公众号。**结尾不使用“后台回复”“菜单栏”“加小编”等运营话术，以内容本身建立信任。**

### 写作流程

1. **先读取本文件** 获取全部写作规范。
2. **查重避雷**：读取 `articles/` 目录下所有文章标题，确保新选题不重复。
3. **选定切入点**：从六大板块中选择一个具体现象或体验。
4. **确定写法**：从六种写法中选一种主导（可混合，但保持主调清晰）。
5. **搜集依据**：若涉及解释，使用科学或心理学常识，仅用于呈现理解而非制造焦虑。
6. **撰写正文**（总字数 1000-3000 字，严格遵循用词与排版规范）。
7. **生成标题**（准备 2-3 个备选，发布时保留一个）。

### 草稿层与模板正文层

- **草稿层**：`## 备选标题`、`## 正文` 标识，发布前必须删除。
- **模板正文层**：从 `# 正式标题` 开始，遵守模板语义。

## 三、写法类型（六种可混合）

一篇好文章通常有一个主导写法，避免全部文章同质化。

| 写法             | 说明                                                       | 适用主题举例                                                |
| :--------------- | :--------------------------------------------------------- | :---------------------------------------------------------- |
| **观察诗意型**   | 用细腻笔触描述一个日常片段，不做过多解释，让读者自行感受。 | “凌晨四点醒来”“睡前被窝”“换床单的夜晚”                      |
| **冷知识解释型** | 从一个有趣现象出发，用科学常识解释其生理或心理机制。       | “睡着时突然抽动”“手指泡水起皱”“打哈欠流泪”                  |
| **体验共鸣型**   | 描述一种普遍但少被谈论的感受，唤起“原来你也这样”的共鸣。   | “午睡醒来的恍惚”“饿过头后的平静”“偏头痛发作时”              |
| **观念重塑型**   | 针对一个常见观念提供新视角，松动非黑即白的认知。           | “分床睡未必感情不好”“睡够8小时不是标准”“疲惫不一定因为缺觉” |
| **读者故事型**   | 以读者来信或征集内容为主体，用他人故事引发共鸣。           | “读者来信：我总是在周日晚上睡不着”“那些在地铁上流泪的人”    |
| **数据洞察型**   | 引用研究报告数据，用数据温柔地消解个体的自责感。           | “48%睡眠困扰率”“你的疲惫可能是缺‘属于自己的时间’”           |

## 四、正文用词原则

### 1. 接纳已发生的温柔逻辑

文章基调应为：“原来如此，身体已经做得很好了。”

- 对已有感受给予理解，而非评判。
- 解释现象时，语气是“原来是这样”，而非“你该怎么做”。

### 2. 确定感与温柔感并存的用词

| 场景         | 推荐用词                     | 避免用词       |
| :----------- | :--------------------------- | :------------- |
| 描述科学事实 | 研究显示、身体会、大脑倾向于 | 你必须、一定要 |
| 描述读者行为 | 很多人会、当你……的时候       | 你不该、你错了 |
| 给出视角     | 可以这样看、另一个角度是     | 正确的做法是   |
| 结尾         | 没关系、身体已经在尽力了     | 从今天开始改变 |

### 3. 禁止词汇清单

- ❌ 绝对承诺类：“保证”“一定”“肯定能”
- ❌ 小编体：“小编”“本宫”“亲们”
- ❌ 过度装饰：连续感叹号、波浪线、大量 emoji（全文 emoji ≤2 处）
- ❌ 运营话术：“后台回复”“点击菜单”“加微信”
- ❌ 焦虑制造类：“你中招了吗”“千万别再”“小心”

## 五、排版细则

### 1. 段落长度控制

| 元素          | 建议字数/行数 |
| :------------ | :------------ |
| 手机屏幕段落  | ≤ 5 行        |
| 单句长度      | ≤ 25 字       |
| `##` 章节间距 | 前后各空 1 行 |
| 段落之间      | 空 1 行       |
| 列表项        | 每项 ≤ 2 行   |

### 2. 加粗规则

- **仅加粗**：关键数据、核心观点、温柔金句。全文 5-8 处。

### 3. 小标题使用

- `##` 章节标题：全文 ≤4 个，每个 4-8 字。
- `###` 副标题：可选。
- `#### 小标题|大标题`：用于切换视角或点出事实，全文 1-2 处。

### 4. 列表与说明卡

- **无序列表**：呈现现象、感受、可能的原因。
- **说明卡 `> `**：放置科学依据或温柔提醒，全文 1-2 处。

## 六、阅读顺序设计

引导读者完成以下心理路径：

1. **“原来你也这样”（共情层）** → 开篇描述一个具体场景或现象。
2. **“原来是这么回事”（认知层）** → 若为解释型写法，给出科学或心理学视角；若为诗意型，则自然过渡到感受层面。
3. **“这样看也挺好”（接纳层）** → 提供温柔新视角，松动原有认知。
4. **“身体/情绪已经尽力了”（归属层）** → 以安慰或开放式结尾收束，不给压力。

## 七、新手期结尾规范

**严禁**：后台回复、菜单点击、加微信、转发福利、任何运营引导。

**允许**：

- ✅ 一句温柔收束：“身体已经在用它能做到的方式照顾你了。”
- ✅ 开放式共鸣：“你走路时会故意避开地砖缝吗？”
- ✅ 简洁留白。

---

## 八、数据与引用标准

- 引用来源：研究报告、生理学常识、心理学理论。
- 格式：`《生理学杂志》一项研究指出……`
- 数据仅用于温柔解释，不用于制造焦虑。
