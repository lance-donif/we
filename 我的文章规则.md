# 阿柒美美屋 · 公众号文章写作规则 V2.3

## 一、账号定位与内容心法

- **核心领域**：美肤、美白、祛黄、护发、食疗养生。
- **目标人群**：30岁以上女性，追求低成本、可持续的变美方式。
- **内容调性**：科学降维、生活化表达、拒绝制造焦虑。
- **核心心法**：将晦涩的医学与护肤理论转化为居家场景中可立即执行的一个具体动作。

---

## 二、选题原则

1. **单一痛点聚焦**：一篇文章仅解决一个具体症状或困惑。
2. **时效关联**：选题需与季节变化、节气更替或节假日后的身体状态恢复直接相关。
3. **门槛可控**：解决方案必须依赖居家常见物品或自然动作，排除昂贵产品与专业设备。
4. **避雷检查**：撰写前需确认该选题在已发布内容中无重复。

---

## 三、标题规则

1. **要素组合**：标题必须同时包含**具体症状描述**与**解决方案核心动作**。
2. **情绪张力**：需包含颠覆常识或制造对比的张力词，但禁用以下低信息量词汇：`震惊`、`揭秘`、`看过来`、`惊呆了`。
3. **字数限制**：标题字数严格控制在 **15字至25字** 区间内。
4. **自检标准**：标题须能让读者在三秒内理解文章将要解决的核心问题，并产生点击欲望。

---

## 四、开头段落规则

> **核心原则：7822法则应用。** 文章前20%的内容决定80%的阅读完成率。开头段落是全文唯一强制执行区域。

### 4.1 物理层硬性约束

- **行数上限**：开头段落（从第一个字符到第一个小标题前）在手机屏幕显示**不得超过5行**。
- **字数密度**：单行字数不超过20字，段落之间空一行以制造呼吸感。

### 4.2 情绪引擎三选一

开头前三行必须且只能选择以下三种情绪引擎之一启动，不可杂糅：

| 引擎类型         | 核心目的                                       | 操作路径描述                                                                           |
| :--------------- | :--------------------------------------------- | :------------------------------------------------------------------------------------- |
| **痛点共鸣引擎** | 让读者产生“这说的就是我”的代入感               | 描述一个具体且带有轻微羞耻感的失败经历或容貌焦虑瞬间。必须使用第一人称。               |
| **认知颠覆引擎** | 打破读者原有认知，制造“原来我之前都错了”的惊讶 | 先陈述一个大众普遍认同的错误做法，紧接着用转折词引出完全相反的底层逻辑。               |
| **好奇缺口引擎** | 用反常现象或反常识结论制造信息差               | 直接抛出一个违反直觉的结果或数字，不急于解释原因，迫使读者为了填补知识缺口而继续阅读。 |

### 4.3 情绪引擎排他性

- 开头段落中，痛点共鸣、认知颠覆、好奇缺口三种引擎**不可同时使用**。
- 若发现两种及以上引擎特征同时出现，必须删除至仅保留一种。

### 4.4 人称与视角强制

- 开头段落**必须包含**第一人称或第二人称，严禁使用第三人称旁观视角。
- 若使用第一人称，必须营造“我是一个走过弯路的亲历者”而非“我是指导你的专家”的平视感。

### 4.5 破冰结束标志

- 在引出第一个小标题之前，必须用一句**加粗短句**对开头进行总结或反转，该短句即为文章的核心论点或价值承诺。

### 4.6 首图诱导布局规则

- **插入位置**：文章开头第一段落结束后，紧接第二段落开始前，必须插入一张**尺寸经过裁剪**的首图。首图不得直接放置于标题下方、正文第一段之前。
- **可见比例控制**：图片在主流手机屏幕上的初始可视高度仅为其总高度的 **15%至20%**。这意味着读者在未滑动屏幕时，只能看到图片的顶部边缘或局部细节，无法看清全貌。
- **视觉引导设计**：图片的可见部分必须包含一个**具有明确指向性的视觉元素**或**不完整的文字截断**，以制造强烈的好奇心缺口，诱使用户主动向下滑动以查看完整内容。
- **图片属性要求**：该首图需符合公众号头条封面图的基本尺寸比例，但内容设计需刻意将核心信息或视觉焦点置于图片的中下部区域。
- **辅助功能**：当开头文字的情绪引擎未能第一时间抓住读者时，首图的部分露出作为第二道防线，利用视觉悬念降低跳出率。

### 4.7 首图位置强制校验

- **禁止位置**：首图不得直接放置于标题下方、正文第一段之前。
- **强制位置**：首图必须插入第一段落与第二段落之间。
- **校验方法**：发布前使用手机预览功能，确认首屏显示时图片仅露出顶部边缘，无法看清全貌。

---

## 五、正文结构规则

1. **递进结构**：全文章节需遵循`认知颠覆 → 原理降维 → 量化动作 → 注意事项`的逻辑顺序。
2. **分段规则**：每个自然段在手机屏幕上显示不超过5行。
3. **章节标识**：章节之间必须使用二级小标题分割，小标题上下各空一行。

---

## 六、干货包装与降维规则

1. **禁止术语裸奔**：任何医学、生物学或化学专业名词出现时，必须紧跟一句将其类比为居家生活场景的描述。
2. **量化指标强制**：任何给出的方法描述中，必须包含**每日执行时长**、**每周执行频次**或**见效周期参考**。
3. **动作颗粒度**：描述操作步骤时，需细化到具体部位、操作方向和力度感受。

---

## 七、配图规范

### 7.1 风格与构图要求

1. **统一风格**：所有配图必须使用暖色调、治愈系猫咪卡通插画。
2. **构图要求**：优先使用多格漫画叙事。图片中出现的任何文字、标签或对话气泡必须使用中文。
3. **拟人化**：鼓励多只不同花色猫咪互动，可穿戴围裙、眼镜等生活化配饰。
4. **情感节拍**：配图需对应所在章节的核心痛点或解决方法，起到阅读过程中的情绪调节作用。

### 7.2 配图数量与位置强制规则

1. **每章必配**：每个章节结束处必须插入一张配图。首图需额外遵循4.6条的特殊布局规则。
2. **最少配图数量**：文章配图总数不得少于 `章节数 + 1`（含首图）。
3. **缺失惩罚**：任何章节结束处若无配图占位符，文章不得进入发布流程。
4. **校验方法**：发布前统计全文图片标记数量，与章节数加一进行对比。

### 7.3 配图提示词格式

1. **占位符格式**：草稿状态中，每个章节结束后必须显式标注配图占位符及提示词。
2. **格式规范**：`![章节配图](图片路径占位符)` 换行 `提示词：[画面主体、叙事细节、中文标签内容、风格标签]`
3. **未标注提示词的配图视为无效占位**。

---

## 八、结尾规范

1. **严禁运营诱导**：结尾禁止出现引导后台回复、点击菜单、加微信、转发福利等任何运营指令。
2. **隐性驱动结构**：结尾需依次完成以下三项隐性任务：
   - **价值观收束**：一句引发读者对文章观点产生认同感的总结性陈述。
   - **可转发金句**：一句可以不依赖全文、独立存在的观点陈述，便于读者直接用于社交分享。
   - **留白共鸣**：一句不把话说完的开放式结尾，触发读者的个人情感补充。
3. **禁用句式**：结尾禁止出现`如果觉得有用请点赞`、`转发给需要的人`、`关注我每天更新`、`你们觉得呢留言告诉我`及其同义变体。

---

## 九、排版规则

1. **段落间距**：段落之间空一行，小标题上下各空一行。
2. **强调方式**：仅使用加粗表示重点，禁用彩色字体与特殊背景色。
3. **引用格式**：引用内容使用引用块格式呈现。
4. **结束标识**：正文结束后使用分隔线标识，其后放置结尾配图与字数统计。

---

## 十、发布前自检清单

### 标题

- [ ] 字数是否在15-25字区间内。
- [ ] 是否包含症状描述与解决方案要素。
- [ ] 是否不含任何禁用词汇。

### 开头（20%重点检查区）

- [ ] 手机预览下前五行是否完成情绪引擎启动。
- [ ] 前三行是否包含第一人称或第二人称。
- [ ] 是否选择了唯一的情绪引擎且未杂糅。
- [ ] 是否在开头结尾处有加粗的核心论点句。
- [ ] 首图是否插入在第一段落与第二段落之间，而非标题正下方。
- [ ] 手机预览下首图初始可见比例是否仅为其总高度的15%至20%。
- [ ] 首图可见部分是否包含指向性视觉元素或文字截断。

### 正文

- [ ] 每个自然段是否不超过5行。
- [ ] 专业名词是否完成生活化降维转化。
- [ ] 方法描述是否包含量化数据。
- [ ] 章节顺序是否遵循`认知颠覆 → 原理降维 → 量化动作 → 注意事项`。

### 结尾

- [ ] 是否不存在任何禁止的运营诱导内容。
- [ ] 是否包含一句可独立转发的金句。
- [ ] 是否以留白收束全文。

### 配图

- [ ] 每个章节后是否配有猫咪卡通插画占位符。
- [ ] 配图提示词是否明确要求中文标签。
- [ ] 配图总数是否达到“章节数 + 1”。
- [ ] 所有配图占位符是否均附有完整提示词。
