# 公众号科普短文写作规则（社会观察/人生复盘/案例分析方向）

## 核心原则：故事育人

无论分析何种社会现象，写作时必须遵循 **「故事育人」**：

- **对我有用**：每一个案例必须能引发读者反思，让读者意识到自己可能犯同样的错误。
- **可操作性**：拒绝空洞说教，用具体案例展示「步步错」的逻辑链，让读者一看就心惊。
- **情绪共鸣**：用第二人称「你」制造代入感，让读者觉得「这说的就是我」。

## 参考文章风格解构（基于《一个女人是如何把路走绝的》）

在撰写此类社会观察文章时，需复刻以下 **「二小姐体」** 核心思路：

### 1. 标题与切入点

- **制造悬念与警示**：提出一个关于人生选择的大问题（如：如何把路走绝？如何避免步步错？）
- **关键词前置**：直接包含「走绝、犯错、勘误」等人生关键决策场景词。
- **反差感**：标题要让人好奇「真的会这样吗？」

### 2. 行文节奏与口语感

- **分段极短**：一段不超过 4 行手机屏幕，多用独立短句。哪怕是完整的人生道理，也要用换行切碎。
- **情绪可视化**：大量使用括号注释表达内心 OS（例：（二小姐敲黑板划重点了！））和网络表情/梗图描述。
- **第二人称对话感**：仿佛在和读者面对面私聊，使用「你」、「大家」、「二小姐告诉你」拉近距离。
- **重复强调**：关键道理要重复两到三遍，用不同表述加强记忆。

### 3. 干货植入逻辑（很关键）

参考文章并非纯故事，而是 **「起手牌分析 - 步步错展示 - 勘误方法论」** 的闭环：

- **痛点前置**：先分析起手牌有多差，设定读者预期。
- **误区反杀**：用加粗大字号否定常见错误（如：**大！错！特！错！情绪价值是很昂贵的，穷人不配吃**）。
- **给出「反常识但真实」的方案**：例如「婚可以乱结，孩子不能乱生」「卖子宫是底层女性最重要的议价筹码」。**干货必须扎心且颠覆认知**。
- **数字量化**：用具体数字展示损失（20万彩礼变58万负债），让读者切实感受到「步步错」的代价。

### 4. 视觉排版与配图暗示

- **图片作为情绪节拍器**：在警示处、转折处、数据展示处插入配图描述（原作是警示图片，我们的新规需**转化为卡通猫风格**以保持合规与一致性）。
- **配图指令保留**：沿用 `CLAUDE.md` 中的提示词规范，但画面内容需对应文中提到的「彩礼、负债、宝妈、网贷」等具体场景。

## 正式写作规范（融合版）

### 一、写作流程

1. **读取本规则**。
2. **查重**：确保选题不与 `articles/` 内历史文章标题撞车。
3. **搭建「二小姐体」大纲**：
   - 开头：抛出「起手牌很差」的分析框架
   - 中段：用具体案例展示「步步错、重复犯错」的过程
   - 结尾：给出「勘误方法论」和情绪升华
4. **撰写正文**（总字数 1000-2000 字，严格控制废话）。
5. **生成备选标题**（2-3个，口语化、带emoji暗示）。

### 二、用词与文风强制标准

- **严禁**：枯燥的「首先、其次、综上所述」。
- **强制**：
  - 每 3-5 段必须出现一次括号吐槽 `（）` 或打破第四面墙的互动。
  - 出现数字时，必须紧跟一句大白话翻译（如：58万负债还无房无业带娃，你特么连零分都没有了）。
  - 否定某个错误动作时，必须使用 **「加粗 + 感叹号 + 换行」** 制造视觉锤（例：**他不配留后代！**）。
  - 关键道理必须重复两遍以上，用不同表述强化记忆。

### 三、猫咪卡通配图规范（适配新场景）

- **风格**：可爱猫咪卡通插画（cute cartoon cat illustration），**四格漫画或场景叙事风格**，暖色调生活化氛围。
- **数量要求**：每篇文章 **3-5张**，分布在起手牌分析处、步步错转折处、方法论说明处。
- **构图核心要素**：
  - **多格叙事**：优先使用 2-4 格漫画形式展现情节发展
  - **场景丰富**：包含室内家具、厨房、房间、工地、办公室等生活化背景细节
  - **多只互动**：2-3 只不同花色的猫咪（橘猫、三花、白猫、黑猫等）互动
  - **拟人化动作**：猫咪站立、穿围裙、戴厨师帽、端盘子、捧东西、拎包、打电话等
  - **情绪表达**：开心、失落、震惊、后悔、释然等丰富表情
  - **文字语言 (强制)**：图片中出现的任何标签、对话、提示文字 **必须为中文**
- **构图适配建议**：
  - 出现「起手牌分析」章节：四格漫画展示猫咪拿到不同大小的牌面，带中文标签「好牌」「烂牌」「烂牌但能翻盘」
  - 出现「彩礼/嫁人」章节：猫咪穿婚纱、拎红包、交换戒指等场景
  - 出现「网贷/负债」章节：猫咪看手机上的贷款数字、抱头痛哭、钱包空空
  - 出现「宝妈/带娃」章节：猫咪妈妈抱小猫咪、喂奶、独自发呆
  - 出现「杀猪盘/诈骗」章节：猫咪对着手机聊天、收到爱心、发现是骗局震惊
  - 出现「勘误方法论」章节：猫咪戴墨镜很酷、手指指向前方、回头喊「跟上」
