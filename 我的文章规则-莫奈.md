# 阿柒美美屋 · 公众号文章写作规则 V2.4

### 写作流程

1. **读取本规则**。
2. **查重**：确保选题不与 `articles/` 内历史文章标题撞车，文章需要保存到 `articles/` 目录下。

---

### 写作的核心：从选题里深刻体现情绪价值

1. **先想清楚**：这篇文章要满足用户的哪种潜意识渴望？（掌控感/希望感/被允许/小确幸/安全感）
2. **这个渴望的本质是什么？** 用户心里真正想听到的是什么？
3. **用自己的话讲出来**：不要用模板，像朋友聊天一样，把你想对用户说的话讲出来

### 文章结构不是固定的

- 不一定非要"认知颠覆→原理降维→量化动作→注意事项"这个逻辑顺序
- 不一定非要四个章节，章节数量可以灵活
- 不一定非要三句话开头，开头方式可以灵活

**但格式规范还是要遵循**：

- 要有备选标题
- 要有目录（不超过4个）
- 要有配图（每章配图，首图位置正确）
- 字数要在1000字以上
- 分段每段不超过5行

**结构是为内容服务的，内容是为情绪价值服务的。**

## 三、用户潜意识渴望与满足方法

### 3.1 目标用户的五种潜意识渴望

女性读美容类文章，背后有五层不易察觉的潜意识渴望。写文章前先确认：你的方法能回应哪一种？

| 渴望层次   | 本质                           | 文章能给的               |
| ---------- | ------------------------------ | ------------------------ |
| **掌控感** | 生活一地鸡毛，至少脸还能自己管 | 一个具体可执行的动作     |
| **希望感** | "还来得及"的心理安慰           | 量化见效周期，给出盼头   |
| **被允许** | 不想内卷但又不想放弃           | 不苛刻的方案，接纳不完美 |
| **小确幸** | 每天变好一点点就够了           | 低成本、居家场景         |
| **安全感** | 怕做错、怕浪费钱               | 容错暗示，不追究结果     |

### 3.2 方法描写的五种心理配方

方法文字不是"告诉用户怎么做"，而是"让用户觉得这次我能做到"。每种渴望对应一种写法：

#### 配方一：满足"掌控感" → 动作要具体到荒唐

用户痛点是"试了很多没用"的挫败感。动作描写必须让脑子里立刻能演出来。

- **❌ 空洞写法**："要坚持护肤，才能改善肤质。"
- **✅ 正确写法**："每天晚上，卸妆之后，用棉柔巾浸冰水，敷在毛孔粗大的地方——不是整张脸，就敷你鼻翼两侧那块，一分钟。"
- **生效原理**：具体到部位、时长、工具，**知道"我能做到"比知道"为什么有用"更重要**。

#### 配方二：满足"希望感" → 量化见效周期

用户焦虑：我是不是已经太晚了？方法里必须给一个"还不算晚"的盼头。

- **❌ 空洞写法**："长期坚持才能看到效果。"
- **✅ 正确写法**："第一周，你会感觉皮肤出油少了。第三周，摸上去会滑一点。第四周，照镜子会想说：哎，好像不太一样了。"
- **生效原理**：把漫长的等待切成可感知的小节点，每个节点都是一次"小赢"，刺激正向循环。

#### 配方三：满足"真人感" → 稍微粗糙一点，像真人写的

文章不要太完美、太官方，要像朋友在聊天。

- **❌ 太精细写法**："将棉柔巾折叠成合适大小，以45度角轻敷于面部，每次1分钟，每日1次。"
- **✅ 粗糙写法**："我自己是这么弄的，随便找张棉柔巾，浸点冰水，往脸上一敷就行，大概敷个几十秒，不用太准。"
- **生效原理**：加入"我自己是这么弄的"、"随便"、"不用太准"这类词，用户会觉得更真实，没有距离感。

#### 配方四：满足"不想内卷" → 方案要"不累"

目标人群讨厌"你必须每天做XX件事"的压迫感。

- **❌ 压迫写法**："每天护肤步骤不能少：水、乳、精华、面霜……"
- **✅ 正确写法**："不需要买新东西。你家厨房里那瓶蜂蜜就行。洗脸之后，手指上搓一点蜂蜜，轻轻按在脸上——就当是给它吃口甜的。五分钟后洗掉。"
- **生效原理**：用"你家本来就有的东西"消除门槛；拟人化表达让动作变得轻松愉快。

#### 配方五：满足"安全感" → 语气要"容错"

不能给用户"万一做坏了怎么办"的顾虑。

- **❌ 风险写法**："一定要按照正确的手法，否则适得其反。"
- **✅ 正确写法**："稍微轻一点重一点都没关系。皮肤又不娇气，它经得住。你搓的时候，它在吸水，方向错了顶多少吸一点，不会坏。"
- **生效原理**：主动说出可能的担忧并化解掉，执行率大幅提升。

#### 配方六：满足"没时间"借口 → 时长要短到离谱

用户最大的借口是"我没时间"。方法要把时长压到低到不可能拒绝。

- **❌ 负担写法**："每晚护肤需要十五到二十分钟。"
- **✅ 正确写法**："每天三分钟。真的三分钟。刷三条短视频的时间。你甚至可以一边刷一边做。"
- **生效原理**：把"时间成本"压到用户无法说"我没空"的程度。

### 3.3 心理配方速查表

| 渴望     | 方法写法        | 关键词             |
| -------- | --------------- | ------------------ |
| 掌控感   | 具体到部位+工具 | "就敷鼻翼两侧"     |
| 希望感   | 节点式量化周期  | "第一周…第三周…"   |
| 真人感   | 稍微粗糙+口语化 | "我自己是这么弄的" |
| 不内卷   | 零成本+轻松语气 | "你家厨房那瓶"     |
| 安全感   | 主动化解担忧    | "搓轻搓重都行"     |
| 时间借口 | 时长短到荒谬    | "三分钟而已"       |

### 3.6 首图诱导布局规则

#### 3.6.1 垂直位置强制约束

- **插入位置**：首图不得放置于标题正下方，也不得放置于第一段落或第二段落之后。
- **强制位置**：首图必须插入在**开头段落群的第五段落与第六段落之间**。
- **开头段落群定义**：从小标题下的第一行正文开始，至首图插入点之前的所有自然段落，共计**不少于三段、不超过四段**。
- **行数累积要求**：首图插入前，开头段落群在手机屏幕上的累积显示行数必须达到 **8至10行**。
- **效果验收标准**：在主流手机预览时，首图仅顶部约 **10%至15%** 的区域可见，主体内容需滑动一次屏幕后方可完整展示。

#### 3.6.2 段落构造要求

- 开头前三段每段不超过3行，以短句为主。
- 第三段末尾需以省略号、破折号或未完结的问句结束，与首图的截断感形成呼应。
- 禁止在首图插入前使用小标题或分割线。

#### 3.6.3 视觉引导要求

- 图片可见边缘区域必须包含**向上延伸的视觉元素**或**横跨截断线的文字笔画**。
- 图片被截断的位置应恰好切在关键信息的中间部位，强化用户向下滑动查看完整的冲动。

### 3.7 首图位置强制校验

- **禁止位置**：首图不得直接放置于标题下方、正文第一段之后、正文第二段之后。
- **强制位置**：首图必须插入第三段落与第四段落之间。
- **校验方法**：发布前使用手机预览功能，确认首屏底部恰好触及图片顶部边缘，图片主体不可见。

## 六、正文结构规则

1. **递进结构**：全文章节需遵循`认知颠覆 → 原理降维 → 量化动作 → 注意事项`的逻辑顺序。
2. **分段规则**：每个自然段在手机屏幕上显示不超过5行。
3. **章节标识**：章节之间必须使用二级小标题分割，小标题上下各空一行。

---

## 七、干货包装与降维规则

1. **禁止术语裸奔**：任何医学、生物学或化学专业名词出现时，必须紧跟一句将其类比为居家生活场景的描述。
2. **量化指标强制**：任何给出的方法描述中，必须包含**每日执行时长**、**每周执行频次**或**见效周期参考**。
3. **动作颗粒度**：描述操作步骤时，需细化到具体部位、操作方向和力度感受。

---

## 八、配图规范

### 8.1 风格与构图要求

1. **统一风格**：所有配图必须使用**印象派油画质感**——笔触柔和、光影朦胧、色彩温暖自然、画面有颗粒感/肌理感。
2. **画面核心**：以**人物**为主，可以是女性、男性或男女一起出现，但不能通篇都是同一种性别。穿着复古/文艺风格服饰，处于自然或生活场景中。
3. **推荐画面元素**：
   - 人物：柔和的侧脸/背影/互动的两人，穿着白衬衫、连衣裙、休闲西装等舒适服饰
   - 场景：阳光草地、花园小径、午后窗边、湖边漫步、厨房做简单食物
   - 细节：发丝在风中飘动、指尖触碰花瓣、手里捧着热饮、脚边有慵懒的猫
4. **核心目的**：图片无需与文章内容强关联，只需在潜意识层面满足用户渴望（掌控感、希望感、放松感、小确幸）。
5. **情感节拍**：配图对应所在章节的情绪基调，用色彩和光影调节阅读心情。

### 8.2 配图数量与位置强制规则

1. **每章必配**：每个章节结束处必须插入一张配图。首图需额外遵循3.6条的特殊布局规则。
2. **最少配图数量**：文章配图总数不得少于 `章节数 + 1`（含首图）。
3. **缺失惩罚**：任何章节结束处若无配图占位符，文章不得进入发布流程。
4. **校验方法**：发布前统计全文图片标记数量，与章节数加一进行对比。

### 8.3 配图提示词格式

1. **占位符格式**：草稿状态中，每个章节结束后必须显式标注配图占位符及提示词。
2. **格式规范**：`![](图片路径占位符)` 换行 `提示词：[人物描述、场景描述、光影氛围、印象派油画质感、满足的渴望类型]`
3. **未标注提示词的配图视为无效占位**。

## 九、结尾规范

1. **严禁运营诱导**：结尾禁止出现引导后台回复、点击菜单、加微信、转发福利等任何运营指令。
2. **隐性驱动结构**：结尾需依次完成以下三项隐性任务：
   - **价值观收束**：一句引发读者对文章观点产生认同感的总结性陈述。
   - **可转发金句**：一句可以不依赖全文、独立存在的观点陈述，便于读者直接用于社交分享。
   - **留白共鸣**：一句不把话说完的开放式结尾，触发读者的个人情感补充。
3. **禁用句式**：结尾禁止出现`如果觉得有用请点赞`、`转发给需要的人`、`关注我每天更新`、`你们觉得呢留言告诉我`及其同义变体。

---

## 十、排版规则

1. **段落间距**：段落之间空一行，小标题上下各空一行。
2. **强调方式**：仅使用加粗表示重点，禁用彩色字体与特殊背景色。
3. **引用格式**：引用内容使用引用块格式呈现。
4. **结束标识**：正文结束后使用分隔线标识，其后放置结尾配图与字数统计。
