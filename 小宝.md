# 公众号科普短文写作规则（女性成长方向）

## 核心原则：写法不要太专业，保持连贯性

### 一、写作流程

1. **读取本规则**。
2. **查重**：确保选题不与 `articles/` 内历史文章标题撞车。
3. **搭建大纲**：
   - 开头：抛出一个关于身体/关系的小烦恼。
   - 中段：科学辟谣 + 给出一个「反直觉妙招」。
   - 结尾：情绪升华 + 单身人士「代餐练习法」。
4. **撰写正文**（总字数 1000-2000 字，严格控制废话）。
5. **生成备选标题**（2-3个，口语化、带emoji暗示）。
