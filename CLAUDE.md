# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 公众号文章写作指南

本文档定义美食养生、美容美发、身心健康等主题公众号文章的写作规范。**每篇文章只讲一个方法，围绕这个方法深入展开，结合心理学原理增加说服力。核心定位：不是教用户变美，而是让用户知道这样做有希望变美。**

### 写作流程

1. **先读取本文件** 获取写作风格要求
2. **查重避雷**：读取 `articles/` 和 `articles/已发布/` 目录下所有文章标题，确保主题不重复
3. **选择一个方法**：从食疗养生、美发护发、美肤护肤、身心调理、生活方式等领域选择一个具体方法（排除已写过的主题）
4. **深入挖掘**：找到这个方法背后的心理学原理、科学依据或传统智慧
5. **撰写正文**（总字数 800-1500 字，使用杂志模板 Markdown 语义）
6. **生成标题**（准备 2-3 个备选）

### 草稿层与模板正文层

公众号文章现在分为两层：

1. **草稿层**
   - 允许保留 `## 备选标题`
   - 允许保留编号标题列表
   - 允许保留 `## 正文`
   - 这些内容属于创作辅助信息，**不属于模板正文结构**

2. **模板正文层**
   - 从首个 `# 正式标题` 开始
   - 必须遵守当前模板语义：
     - `# 主标题`
     - `## 章节标题`
     - `### 章节副标题`
     - `#### 小标题|大标题`
     - `!! 高亮事实块`
     - `@@ 大卡片式方案块`
      - 正常段落
      - 无序列表
      - `> ` 说明卡
      - 连续图片图组

3. **人工编辑层**
   - 顶部封面图
   - 备选标题筛选后的最终标题确认
   - 这些内容可以保留在写作草稿里，但**不属于代码必须识别的正文语义**

**重要：当前代码不会自动忽略草稿层，也不会专门识别人工编辑层。**

- `## 备选标题`
- `## 正文`
- `[封面图提示词：...]`
- `[图片提示词：...]`

这些都只是写作辅助信息。进入正式模板预览、复制发布或交付最终正文前，必须删除或移出模板正文。

### 文章类型

每篇文章只聚焦**一个具体方法**，可以来自以下领域：

**领域类型：**

1. **食疗养生** — 某种食材/食谱的具体功效
2. **美发护发** — 某个护发方法/技巧
3. **美甲技巧** — 某个指甲护理方法
4. **美肤护肤** — 某个护肤步骤/方法

7. **生活方式** — 睡眠改善、习惯养成等

### 写作要求

#### 字数

- 总字数 800-1500 字（不含标题）

#### 核心定位

**不是教用户变美，而是让用户知道这样做有希望变美**

- 重点传递"可能性"和"希望"，而非绝对的"变美承诺"
- 用词要温和：可以、可能、有助于、或许能，而非绝对化的词语
- 强调个体差异：每个人的体质、情况不同，效果因人而异
- 给用户信心：不是承诺效果，而是提供方向和希望
- 鼓励尝试：告诉用户"值得一试"，而不是"必须这样做"

#### 心理学原理运用

**通过暗示心理学原理增加说服力：**

1. **社会认同原理**
   - 提到"很多人试过"、"越来越多的人开始..."
   - 例："很多人坚持一个月后发现..."

2. **权威效应**
   - 引用研究、专家观点、传统智慧
   - 例："中医认为..."、"营养学研究发现..."
   - 暗示科学性，但不要过于学术

3. **稀缺性原理**
   - 强调方法的独特性或时间窗口
   - 例："春季是最佳时机"、"这个小技巧很少人知道..."

4. **一致性原理**
   - 引导用户从小行动开始，建立承诺感
   - 例："只需要5分钟"、"从今天开始..."

5. **希望效应**
   - 给用户期待和信心
   - 例："也许一个月后，你会看到不一样的自己"
   - 用案例或可能性暗示效果，但不保证

6. **锚定效应**
   - 用对比突出方法的简单性
   - 例："不需要花大钱"、"比想象中简单"

**运用技巧：**
- 不要直接说"心理学研究表明"，而是自然融入叙述中
- 用故事、案例、对比等方式暗示原理
- 让用户自己得出"这个方法值得一试"的结论

#### 风格

- 亲切自然，像闺蜜分享心得
- 语言轻松活泼，通俗易懂
- 适合年轻女性读者阅读
- 传递积极正面的能量，给用户希望和动力
- 多用"你"拉近距离，增强互动感
- 参考圆周旅迹杂志风：章节化叙事、克制的色彩节奏、偏纸张感的阅读氛围、少量 emoji 点缀气氛

#### 核心写作手法

**四大写作手法，每次只选一个重点运用：**

1. **痛点破局法**（解决"为什么看"的问题）
2. **情绪共鸣法**（解决"信任感"的问题）
3. **认知减负法**（解决"看不下去"的问题）
4. **价值锚定法**（解决"凭什么转/赞"的问题）

**重要：一篇文章只重点运用一个手法，其他手法可以自然融入，但不要全部堆砌。**

#### 标题风格

- 准备 2-3 个吸引力标题供选择
- **固定格式：你真美 | XXXX内容**
- 要求：实用性强、有好奇心、能解决痛点
- 突出单一方法的特点

**标题示例：**
- ✅ 你真美 | 每天冥想5分钟，气色竟然变好了？
- ✅ 你真美 | 不花钱的美容秘籍：冥想比护肤品更管用
- ✅ 你真美 | 为什么越来越多的人开始冥想？

#### 文章结构

**一篇完整的文章仍然遵循五段式逻辑，但要落地到 4-6 个 `##` 章节中：**

1. **引入痛点**（100-200字）
   - 描述用户面临的问题
   - 引发共鸣，让用户觉得"这就是我"
   - 用 `##` + `###` 组织，不再用 `---` 强行分段

2. **介绍方法**（200-400字）
   - 具体步骤和操作方法
   - 简单易懂，可操作性强
   - 重点方法入口可用 `#### 小标题|大标题`

3. **心理学原理/科学依据**（200-300字）
   - 自然融入心理学原理或科学依据
   - 解释为什么这个方法可能有效
   - 可以穿插 1 组图片或 1 个说明卡增强阅读节奏

4. **真实案例/可能性描述**（150-250字）
   - 描述可能的改变（用"很多人"、"也许"等词）
   - 给用户期待和希望
   - 强调个体差异
   - 可以搭配列表卡或说明卡强化重点

5. **鼓励行动**（100-150字）
   - 降低尝试门槛
   - 给出具体行动建议
    - 温和的结语
   - 最后一章仍然用 `##` / `###` 收束全文

**重要：避免每篇都用相同的子标题，要根据主题灵活变化。**

#### 模板语义建议

- 每篇文章正文建议 4-6 个 `##` 章节
- 每篇文章建议使用 1-3 处 `#### 小标题|大标题`
- 每篇文章建议使用 1-3 处 emoji 点缀，参考 `id=10` / `id=16` 的杂志稿气氛
- emoji 优先放在开篇氛围行、节奏转折处或说明卡中，不要堆进主标题和每个列表项
- 无序列表默认会被渲染成浅色信息卡，适合放步骤、提醒、变化、注意事项
- `> ` 默认会被渲染成说明卡，适合放结论、提醒、边界条件
- `!! ` 默认会被渲染成高亮事实块，适合放关键一句、经验钉子句、读者提醒
- `@@ ` 默认会被渲染成大卡片式方案块，适合放推荐方案、同款步骤、适用场景、懒人版总结
- 正文图片不是每段都要有，通常每篇安排 2-3 组图组即可

#### 模板风格参考

- `杂志 · 探洞`：灰白纸感 + 青绿/橙/绿轮换，适合方法拆解、成分解释、理性护肤题材
- `杂志 · 苗寨`：浅绿纸感 + 黄橙辅助，适合温和调理、生活方式、身体状态改善题材
- `杂志 · 巴尔干`：灰纸 + 橙绿节奏，适合有对比、有决心感、需要“别再拖了”这种情绪的题材
- `杂志 · 河北`：灰/青/黄/蓝古城感，适合结构清楚、知识点分层、需要多段节奏变化的题材
- `杂志 · 锦囊`：蓝青系提示感，适合经验总结、避坑技巧、提醒类题材
- `杂志 · 新疆`：灰白 + 冰蓝节奏，适合强调氛围、季节、环境感的题材
- `杂志 · 长沙`：橙蓝粉黄对比更强，适合热闹、生活流、消费决策、场景体验类题材

#### 参考稿模块映射

- 参考稿的大号章节编号与主标题块，对应本项目的 `## 章节标题`
- 参考稿的章节副标题与情绪引子，对应 `### 章节副标题` 或少量 `### emoji 氛围行`
- 参考稿的小型主题引出块，对应 `#### 小标题|大标题`
- 参考稿的浅色说明卡，对应 `> `
- 参考稿的信息整理块，对应无序列表
- 参考稿的单句高亮提示块，对应 `!! 高亮事实块`
- 参考稿的大型推荐/同款卡片，对应 `@@ 大卡片式方案块`
- 参考稿的并排图片/单图停顿，对应连续 `![](...)`

#### 封面图说明

- 顶部封面图继续保留为写作与排版阶段的人工编辑动作
- 如需封面图，位置仍然放在 `# 主标题` 下方，比例建议 `2.35:1`
- 但封面图不计入“代码必须识别的正文语义”，也不参与参考稿模块对齐判断

#### 模块密度硬规则

- 每篇文章必须有 4-6 个 `##` 章节，不能少于 4 个
- 每篇文章至少 2 个章节带强化模块
- 强化模块只能从以下几类中选择：`#### 小标题|大标题`、无序列表、`> `、`!! 高亮事实块`、`@@ 大卡片式方案块`、连续图片图组、1 行 emoji 氛围行
- 不允许每个章节都堆强化模块，至少要有 1 个章节只靠文字成立
- 中段章节必须至少出现 1 次 `#### 小标题|大标题`
- 全文至少出现 1 次无序列表、`> `、`!! `、`@@ ` 中的任意一种，不能全部没有
- `!! 高亮事实块` 每篇建议 1-2 次，最多 3 次
- `@@ 大卡片式方案块` 每篇建议 0-1 次，最多 1 次

#### 图片插入硬规则

- 首章允许 0-1 张图，但必须放在一个完整段落之后，不能放在第一段前面
- 中段的方法/原理章节至少有 1 组图，优先放在一个完整信息块之后，而不是段落中间
- 变化/案例章节优先放 2 张并排图，形成状态呼应或场景呼应
- 收尾行动章节原则上不放图，保持结尾干净
- `#### 小标题|大标题` 前面不要直接插图，至少先有一段说明或一个信息块
- `@@ 大卡片式方案块` 优先放在一个章节尾部，作为收束，不要夹在解释段中间
- `!! 高亮事实块` 优先放在段落之后，用来钉住结论，不要连续堆叠两个
- 纯解释型长段首句前不要插图，先立住认知再给图
- 如果一篇文章只有 1 组图，优先放在中段方法章节，而不是首章或结尾

#### emoji 使用建议

- 可以单独成一行，作为氛围型 `###`，例如：`### 🌼 🍯 🌿`
- 可以少量放进正文或说明卡中，增强生活感和杂志感
- 优先使用和主题有关的自然类、食材类、情绪类 emoji
- 一篇文章控制在 1-3 组，不要过度装饰
- 不要在 `# 主标题` 中使用 emoji，避免标题显得轻浮

### 文章格式规范

1. **分段清晰**：每个部分用空行分隔
2. **重要信息加粗**：关键步骤、时间、用量单独突出
3. **分步骤说明**：优先用无序列表承载步骤与提醒，让模板渲染成信息卡
4. **重点入口**：重要方法点可使用 `#### 小标题|大标题`
5. **关键钉子句**：适合单独强调的一句话可以写成 `!! 关键句`
6. **方案总结卡**：需要图+标题+元信息+描述的总结块，使用 `@@ 标签|标题|时间/数量|简短说明|图片地址`
7. **不要在正文中使用 `---` 分隔段落**
8. **模板正文必须从 `# 主标题` 开始**

### 高亮事实块与方案块语法

如果需要补充参考稿里的“单句高亮提示块”和“大型推荐卡片”，请使用以下固定语法：

```markdown
!! 这是一句要被单独强调的关键提醒或经验钉子句

@@ 推荐方案|标题|时间/数量|简短说明|图片地址
```

规则如下：

1. `!! ` 用于单句高亮事实块
2. `@@ ` 用于大卡片式方案块
3. `@@ ` 后面固定 5 个字段，用 `|` 分隔：
   - 标签
   - 标题
   - 时间/数量
   - 简短说明
   - 图片地址
4. `@@ ` 不要放在章节开头第一段前面，优先放在章节尾部做总结
5. `!! ` 不要连续写两条，避免过度强调

### 图片占位与提示词规范

如果某个位置需要图片，请使用以下固定写法：

```markdown

[图片提示词：如果这是正文图，这里写清楚画面主体、场景、构图、光线、风格、氛围、色调]
```

**图片提示词必须包含以下要素：**

1. **人物描述**（如有人物）：
   - 年龄、性别、种族
   - 五官特征（双眼皮、眼睛大小、颧骨、脸型等）
   - 妆容风格（自然、清透、淡粉腮红等）
   - 发型发色
   - 肤质描述（白皙、瓷肌、毛孔可见、纹理真实等）

2. **服装道具**：
   - 具体服装（颜色、材质、款式）
   - 配饰（帽子、围巾、口罩等）
   - 与主题相关的道具

3. **场景环境**：
   - 具体场景（影棚、咖啡馆、公园、卧室等）
   - 背景细节（无缝背景、窗台、家具等）
   - 季节氛围（春季柳絮、冬季雪花等）

4. **光线与色调**：
   - 光线类型（自然光、柔光、侧逆光、影棚灯光等）
   - 光线角度和强度
   - 整体色调（暖白、淡粉、米色等）
   - 空气感、通透感描述

5. **技术参数**：
   - 镜头焦距（85mm人像、微距镜头等）
   - 光圈大小（f/1.8-2.8等）
   - 景深效果（背景虚化、景深适中等）
   - 画质要求（真实肌肤纹理、高端杂志品质等）

6. **构图要求**：
   - 画面比例（封面图2.35:1、正文图纵向/横向）
   - 构图方式（人物居中、留白、三分法等）
   - 视角（平视、俯视、仰视等）

7. **风格氛围**：
   - 整体风格（杂志摄影、生活方式、插画风格等）
   - 情绪氛围（温暖、宁静、清新、治愈等）
   - 画面感受（自然、专业、信任感等）

### 子标题设计指南

**设计原则：**
- 与文章主题契合
- 与写作手法匹配
- 有变化有节奏
- 引发读者好奇

**设计方法：**
1. 看文章主题是什么
2. 看运用了哪个写作手法
3. 从对应的子标题变体库中选择
4. 根据具体内容微调

**子标题变体库：**

**第一部分：引入痛点/原理**
- 痛点破局法：你的XX为什么总是XX？/ XX的真正原因可能不是你想的那样 / 问题可能不在XX，而在XX
- 情绪共鸣法：我特别理解你的感受 / 这种焦虑，我懂 / 你不是一个人在战斗
- 价值锚定法：为什么XX能帮你？/ XX背后的科学依据 / 这个方法为什么有效？
- 认知减负法：XX其实很简单 / 只需要3步 / 一学就会的XX方法

**第二部分：具体方法**
- 操作类：具体怎么做？/ 3步搞定XX / 跟着做就行 / 简单3步，一看就会
- 饮食类：怎么吃效果最好？/ 这样吃，吸收更好 / 每天这样安排 / 吃对时间很重要
- 生活方式类：从今天开始养成习惯 / 每天给自己几分钟 / 把XX融入生活

**第三部分：效果描述**
- 效果类：坚持两周后 / 一个月后的变化 / 很多人发现了改变 / 效果因人而异，但值得一试
- 案例类：有人这样分享 / 真实体验是这样的 / 看看他们的故事
- 对比类：做和不做，差别在这里 / 坚持vs放弃，结果截然不同

**第四部分：鼓励行动**
- 行动类：今晚就开始吧 / 给自己一个机会 / 不妨试一试 / 从今天开始改变
- 激励类：改变从现在开始 / 你值得更好的自己 / 相信时间的力量
- 轻松类：试试看，反正不花钱 / 不试试怎么知道没用？ / 从小事做起

**示例组合：**
- 黄瓜敷脸（实操类）+ 价值锚定法：为什么黄瓜能补水？→ 怎么敷？给你3个步骤 → 坚持一周后 → 今晚就开始吧
- 冥想减压（身心调理）+ 情绪共鸣法：这种焦虑，我懂 → 只需要5分钟 → 很多人都发现了改变 → 给自己一个机会
- 早睡去黑眼圈（痛点类）+ 痛点破局法：黑眼圈为什么总是消不掉？→ 从今晚开始改变 → 两周后的变化 → 试试看，也许会有惊喜
