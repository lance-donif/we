# 公众号科普短文写作规则（亲密关系/两性/生活干货方向）

## 核心原则：利我主义

无论选题多么猎奇或日常，写作时必须遵循 **「利我主义」**：

- **对我有用**：每一段干货必须能直接应用于读者的真实生活场景（如：如何改善口臭、如何提升体验）。
- **可理解性**：拒绝学术黑话，用「人话」解释生理机制，让读者一看就懂，懂了就能用。

## 参考文章风格解构（基于《男女接吻时，谁更喜欢伸舌头？》）

在撰写此类生活科普时，需复刻以下 **「小薇体」** 核心思路：

### 1. 标题与切入点

- **制造悬念与反差**：提出一个具体的、带有亲密关系摩擦的问题（如：谁更爱伸舌头？为什么中年夫妻不亲了？）。
- **关键词前置**：直接包含「舌吻、口臭、接吻」等高搜索量场景词。

### 2. 行文节奏与口语感

- **分段极短**：一段不超过 4 行手机屏幕，多用独立短句。哪怕是完整的科学论述，也要用换行切碎。
- **情绪可视化**：大量使用括号注释表达内心 OS（例：（没有说女性口臭能容忍的意思！））和网络表情/梗图描述。
- **第二人称对话感**：仿佛在和闺蜜/兄弟私聊，使用「你」、「大家」、「小薇我啊」拉近距离。

### 3. 干货植入逻辑（很关键）

参考文章并非纯科普，而是 **「痛点认知 - 破除误区 - 反常识解法」** 的闭环：

- **痛点前置**：先聊口臭、担心传染病（读者内心顾虑）。
- **误区反杀**：用加粗大字号否定常见错误（如：大！错！特！错！舌根催吐）。
- **给出「非主流但有效」的方案**：例如「用舌背亲避开口臭」、「舔酸奶练习法」。**干货必须出人意料且具体可操作**。

### 4. 视觉排版与配图暗示

- **图片作为情绪节拍器**：在尴尬处、转折处、爆笑处插入配图描述（原作是表情包，我们的新规需**转化为卡通猫风格**以保持合规与一致性）。
- **配图指令保留**：沿用 `CLAUDE.md` 中的提示词规范，但画面内容需对应文中提到的「舌头、酸奶、冰箱」等具体物件。

## 正式写作规范（融合版）

### 一、写作流程

1. **读取本规则**。
2. **查重**：确保选题不与 `articles/` 内历史文章标题撞车。
3. **搭建「小薇体」大纲**：
   - 开头：抛出一个关于身体/关系的小烦恼。
   - 中段：科学辟谣 + 给出一个「反直觉妙招」。
   - 结尾：情绪升华 + 单身人士「代餐练习法」。
4. **撰写正文**（总字数 1000-2000 字，严格控制废话）。
5. **生成备选标题**（2-3个，口语化、带emoji暗示）。

### 二、用词与文风强制标准

- **严禁**：枯燥的「首先、其次、综上所述」。
- **强制**：
  - 每 3-5 段必须出现一次括号吐槽 `（）` 或打破第四面墙的互动。
  - 出现科普数据时，必须紧跟一句大白话翻译。
  - 否定某个错误动作时，必须使用 **「加粗 + 感叹号 + 换行」** 制造视觉锤（例：**大！错！特！错！**）。

### 三、猫咪卡通配图规范（适配新场景）

- **风格**：猫咪卡通（cute cartoon cat illustration）。
- **数量要求**：每篇文章 **3-5张**，分布在痛点场景、转折爆笑处、方法说明处。
- **构图适配建议**：
  - 出现「口臭」章节：猫咪叼着牙刷、猫咪闻酸奶。
  - 出现「舌吻技巧」章节：两只猫咪碰鼻尖、猫咪舔毛绒球。
  - 出现「狐臭/体味」章节：猫咪捂鼻子、猫咪喷香水/除味喷雾、猫咪闻味道做判断。
  - 出现「开口说」章节：两只猫咪头靠头小声说话、一只猫递纸条。
  - 出现「方法/改善」章节：猫咪戴墨镜很酷、猫咪吃蔬菜水果、猫咪打针很怂。
- **提示词格式**（严格执行）：

  ```markdown
  ![](../public/articles/img/cat-kiss-nose.png)

  提示词：两只圆脸猫咪轻轻触碰鼻尖，其中一只害羞地闭着眼，粉红柔光背景，猫咪卡通插画，温馨治愈
  ```
