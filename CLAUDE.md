## 公众号文章写作指南

本文档定义美食养生、美容美发、身心健康等主题公众号文章的写作规范。**每篇文章只讲一个方法，围绕这个方法深入展开，结合心理学原理增加说服力。核心定位：不是教用户变美，而是让用户知道这样做有希望变美。**

### 写作流程

1. **先读取本文件** 获取写作风格要求
2. **查重避雷**：读取 `articles/` 和 `articles/已发布/` 目录下所有文章标题，确保主题不重复
3. **选择一个方法**：从食疗养生、美发护发、美肤护肤、身心调理、生活方式等领域选择一个具体方法（排除已写过的主题）
4. **深入挖掘**：找到这个方法背后的心理学原理、科学依据或传统智慧
5. **撰写正文**（总字数 800-1500 字，使用杂志模板 Markdown 语义）
6. **生成标题**（准备 2-3 个备选）

### 草稿层与模板正文层

公众号文章现在分为两层：

1. **草稿层**
   - 允许保留 `## 备选标题`
   - 允许保留编号标题列表
   - 允许保留 `## 正文`
   - 这些内容属于创作辅助信息，**不属于模板正文结构**

2. **模板正文层**
   - 从首个 `# 正式标题` 开始
   - 必须遵守当前模板语义：
     - `### 主标题`
     - `## 章节标题`
     - `### 章节副标题`
     - `#### 小标题|大标题`
     - 正常段落
     - 无序列表
     - `> ` 说明卡

**重要：当前代码不会自动忽略草稿层。**

- `## 备选标题`
- `## 正文`

这些都只是写作辅助信息。进入正式模板预览、复制发布或交付最终正文前，必须删除或移出模板正文。

每篇文章只聚焦**一个具体方法**，可以来自以下领域：

**领域类型：**

1. **食疗养生** — 某种食材/食谱的具体功效
2. **美发护发** — 某个护发方法/技巧

### 写作要求

#### 字数

- 总字数 800-4000 字（不含标题）
- **段落控制：全文最多 4 段（以 `##` 计算的章节），根据实际内容调整，不必每次都写满 4 段**
- **不要过度拆分段落，保持内容紧凑流畅**

#### 核心定位

**不是教用户变美，而是让用户知道这样做有希望变美**

- 重点传递"可能性"和"希望"，而非绝对的"变美承诺"
- 用词要温和：可以、可能、有助于、或许能，而非绝对化的词语
- 强调个体差异：每个人的体质、情况不同，效果因人而异
- 给用户信心：不是承诺效果，而是提供方向和希望
- 鼓励尝试：告诉用户"值得一试"，而不是"必须这样做"

#### 心理学原理运用

**通过暗示心理学原理增加说服力：**

1. **社会认同原理**
   - 提到"很多人试过"、"越来越多的人开始..."
   - 例："很多人坚持一个月后发现..."

2. **权威效应**
   - 引用研究、专家观点、传统智慧
   - 例："中医认为..."、"营养学研究发现..."
   - 暗示科学性，但不要过于学术

3. **稀缺性原理**
   - 强调方法的独特性或时间窗口
   - 例："春季是最佳时机"、"这个小技巧很少人知道..."

4. **一致性原理**
   - 引导用户从小行动开始，建立承诺感
   - 例："只需要5分钟"、"从今天开始..."

5. **希望效应**
   - 给用户期待和信心
   - 例："也许一个月后，你会看到不一样的自己"
   - 用案例或可能性暗示效果，但不保证

6. **锚定效应**
   - 用对比突出方法的简单性
   - 例："不需要花大钱"、"比想象中简单"

**运用技巧：**

- 不要直接说"心理学研究表明"，而是自然融入叙述中
- 用故事、案例、对比等方式暗示原理
- 让用户自己得出"这个方法值得一试"的结论

#### 风格

- 亲切自然，像闺蜜分享心得
- 语言轻松活泼，通俗易懂
- 适合年轻女性读者阅读
- 传递积极正面的能量，给用户希望和动力
- 多用"你"拉近距离，增强互动感
- 参考圆周旅迹杂志风：章节化叙事、少量 emoji 点缀气氛

#### 核心写作手法

**四大写作手法，每次只选一个重点运用：**

1. **痛点破局法**（解决"为什么看"的问题）
2. **情绪共鸣法**（解决"信任感"的问题）
3. **认知减负法**（解决"看不下去"的问题）
4. **价值锚定法**（解决"凭什么转/赞"的问题）

**重要：一篇文章只重点运用一个手法，其他手法可以自然融入，但不要全部堆砌。**

#### 标题风格

- 准备 2-3 个吸引力标题供选择
- **口语化疑问句风格，像朋友之间的解释或好奇探讨**
- 要求：有好奇心、自然亲切、像在解答一个疑问
- 突出单一方法的特点

**标题写法参考：**

- ✅ 你真美 | 为什么泡脚能帮你睡着
- ✅ 你真美 | 为什么越累越睡不着
- ✅ 你真美 | 为什么你总是翻来覆去
- ✅ 你真美 | 为什么早睡反而让你更精神
- ✅ 你真美 | 为什么蜂蜜水能帮你抗过敏

#### 文章结构

**文章不分太多章节，根据内容自然流动，最多 4 段（段落 = 两个空行分隔的段落块，不是 `##` 章节）。**

1. **文章开头直接进入故事/痛点**
   - **正文开篇不要加任何章节标题**，紧贴主标题后面开始
   - 描述用户面临的问题，用具体的小场景切入
   - 引发共鸣，让用户觉得"这就是我"

2. **自然过渡到方法和原理**
   - 不需要用 `##` 标题切割，根据内容自然流动
   - 融入心理学原理或科学依据
   - 可以穿插 1 个说明卡 `> ` 或重点入口 `#### 小标题|大标题`

3. **真实案例/可能性描述**
   - 描述可能的改变（用"很多人"、"也许"等词）
   - 给用户期待和希望，强调个体差异

4. **鼓励行动（可选）**
   - 如果内容已经完整，可以不单独成段
   - 如需收尾，降低尝试门槛，温和结语

**重要规则：**
- 正文开篇不要加任何标题，直接进入故事
- 全文最多 4 段（两个空行分隔的段落块），根据实际内容调整，不必写满
- 不要过度拆分内容，保持文章的整体性和流动性

**正确示例（早睡去黑眼圈风格）：**

```
# 你真美 | 每天早睡一小时，黑眼圈竟然淡了

每天早上照镜子，是不是总被那双熊猫眼吓到？明明遮瑕膏涂了一层又一层，黑眼圈还是若隐若现。你以为是睡眠不足，于是周末补觉到中午，结果黑眼圈依然纹丝不动。

我特别理解这种绝望。黑眼圈这东西，比顽固的前任还难甩掉。

但问题可能不在你睡了多久，而在你几点睡。

大多数人的错误做法：平时熬夜到凌晨，周末睡到中午补觉。为什么？因为身体的修复有固定的时间窗口……

（全文自然流动，不要频繁用 ## 标题切割段落）
```

#### 模板语义建议

- 每篇文章正文建议 4-6 个 `##` 章节
- 每篇文章建议使用 1-3 处 `#### 小标题|大标题`
- 每篇文章建议使用 1-3 处 emoji 点缀，参考 `id=10` / `id=16` 的杂志稿气氛
- emoji 优先放在开篇氛围行、节奏转折处或说明卡中，不要堆进主标题和每个列表项
- 无序列表默认会被渲染成浅色信息卡，适合放步骤、提醒、变化、注意事项
- `> ` 默认会被渲染成说明卡，适合放结论、提醒、边界条件

#### emoji 使用建议

- 可以单独成一行，作为氛围型 `###`，例如：`### 🌼 🍯 🌿`
- 可以少量放进正文或说明卡中，增强生活感和杂志感
- 优先使用和主题有关的自然类、食材类、情绪类 emoji
- 一篇文章控制在 1-3 组，不要过度装饰
- 不要在 `# 主标题` 中使用 emoji，避免标题显得轻浮

### 子标题设计指南

**设计原则：**

- 与文章主题契合
- 与写作手法匹配
- 有变化有节奏
- 引发读者好奇

**设计方法：**

1. 看文章主题是什么
2. 看运用了哪个写作手法
3. 从对应的子标题变体库中选择
4. 根据具体内容微调

**子标题变体库：**

**第一部分：引入痛点/原理**

- 痛点破局法：你的XX为什么总是XX？/ 你其实是XX / 问题可能不在XX，而在XX
- 情绪共鸣法：我特别理解你的感受 / 这种焦虑，我懂 / 你不是一个人在战斗
- 价值锚定法：为什么XX能帮你？/ XX背后的科学依据 / 这个方法为什么有效？
- 认知减负法：XX其实很简单 / 只需要3步 / 一学就会的XX方法

**第二部分：具体方法**

- 操作类：具体怎么做？/ 3步搞定XX / 跟着做就行 / 简单3步，一看就会
- 饮食类：怎么吃效果最好？/ 这样吃，吸收更好 / 每天这样安排 / 吃对时间很重要
- 生活方式类：从今天开始养成习惯 / 每天给自己几分钟 / 把XX融入生活

**第三部分：效果描述**

- 效果类：坚持两周后 / 一个月后的变化 / 很多人发现了改变 / 效果因人而异，但值得一试
- 案例类：有人这样分享 / 真实体验是这样的 / 看看他们的故事
- 对比类：做和不做，差别在这里 / 坚持vs放弃，结果截然不同

**第四部分：鼓励行动**

- 行动类：今晚就开始吧 / 给自己一个机会 / 不妨试一试 / 从今天开始改变
- 激励类：改变从现在开始 / 你值得更好的自己 / 相信时间的力量
- 轻松类：试试看，反正不花钱 / 不试试怎么知道没用？ / 从小事做起

**示例组合：**

- 黄瓜敷脸（实操类）+ 价值锚定法：为什么黄瓜能补水？→ 怎么敷？给你3个步骤 → 坚持一周后 → 今晚就开始吧
- 冥想减压（身心调理）+ 情绪共鸣法：这种焦虑，我懂 → 只需要5分钟 → 很多人都发现了改变 → 给自己一个机会
- 早睡去黑眼圈（痛点类）+ 痛点破局法：黑眼圈为什么总是消不掉？→ 从今晚开始改变 → 两周后的变化 → 试试看，也许会有惊喜
